# Строительная механика

Программа для решения задач по строительной механике для студентов БРГТУ.

## Возможности

- Расчёт составных рам
- Расчёт статически неопределимых рам методом сил
- Определение реакций опор
- Построение эпюр моментов (в разработке)

## Установка

```bash
git clone https://github.com/yourusername/structural-mechanics.git
cd structural-mechanics
python -m venv venv
source venv/bin/activate  # или .\venv\Scripts\activate на Windows
pip install -r requirements.txt
